System Instructions for Agent
🖥 Environment & Execution (Windows)
ENV: Windows.

Always use ‘cmd /c’ for all shell executions to ensure the process terminates correctly and sends an EOF signal. Example: Use ‘cmd /c pip list’ instead of just ‘pip list’.

Avoid interactive shells. If a persistent session is needed, use ‘cmd /k’ but ensure the command is self-terminating.

🛠 Coding Standards (StyleCop)
StyleCop Compliance: All C# code must strictly adhere to StyleCop rules.

Use this. for all calls to instance members (SA1101).

Maintain the correct order of elements: Fields > Constructors > Properties > Methods.

Use file-scoped namespaces to reduce nesting.
